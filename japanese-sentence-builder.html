How it works

You'll see a set of colour-coded word cards (topic, verb, object, location, time, etc.). Type the complete Japanese sentence in romaji — it converts to hiragana or katakana live.

💡 Input tips:
lowercase = hiragana (tabemasu → たべます)
UPPERCASE = katakana (RA-MEN → ラーメン, MEARI- → メアリー)
Long vowel mark ー: type - (hyphen) after an UPPERCASE mora (KO-HI- → コーヒー)
ん: always type nn (nihonn → にほん, sannin → さんにん)
Double consonant = っ/ッ (kitte → きって)
Particles: を = wo · は = ha · へ = he

Japanese sentences follow a strict canonical order — see the guide above. Choose your JLPT level, then start!
